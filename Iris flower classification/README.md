# 🌸 Iris Flower Classification using Machine Learning

A professional Machine Learning web application that predicts the species of an Iris flower based on its sepal and petal measurements. This project was developed as part of the **DecodeLabs Artificial Intelligence Internship - Project 2**.

---

## 🚀 Live Demo

🔗 **App Link:** * https://iris-flower-classification-7bnawqtdqbuwrg6txmhyxa.streamlit.app/

## 📌 Features

- 🌸 Predict Iris flower species
- 📊 Interactive Streamlit Web Interface
- 📁 Dataset Preview
- 📈 Model Accuracy Display
- 🎛️ User-friendly Sidebar Inputs
- 🎯 Real-time Prediction
- 📋 Prediction Probability
- 🎨 Modern Responsive UI

---

## 🧠 Machine Learning Algorithm

- Random Forest Classifier

---

## 📂 Dataset

The project uses the famous **Iris Dataset** containing 150 flower samples.

### Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Target Classes

- Iris-setosa
- Iris-versicolor
- Iris-virginica

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/ganeshkumar1887/iris-flower-classification.git
```

Go to project folder

```bash
cd iris-flower-classification
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
