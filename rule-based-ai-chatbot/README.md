# 🤖 Rule-Based AI Chatbot

A simple Rule-Based AI Chatbot developed using Python and Streamlit as part of the **DecodeLabs Artificial Intelligence Internship - Project 1**.

## 🚀 Features

- 👋 Greeting Responses
- 💬 Rule-Based Conversation
- 📅 Date & Time
- 🧮 Calculator
- 😂 Jokes
- 📖 Help Menu
- 🎨 Modern Streamlit UI

## 🛠️ Technologies Used

- Python
- Streamlit

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
